Uso do Container - Bootstrap

O container pode ser definido como caixa que ocupam o espaço de uma tela. No bootstrap eles são seletores de clase que podem ocupar uma largura fixa responsiva, neste caso o seletor de classe container ocupa a área central da tela com margens laterais a esquerda e a direita

Nesta duv estamis usando o seletor de classe container.

Uso do container-fluid

O container-fluid como dito anteriormente ocupa 100% da tela, e não aplica margins a direita ou à esquerda.
