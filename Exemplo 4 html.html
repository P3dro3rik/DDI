Exemplo 4 HTML - Estrutura geral de um documentoHTML

Este exemplo mostra a estrutura minima de um documento html. Aqui estão as principais tags utilizadas:

  • <!DOCTYPE html >: Define o tipo de documento html, Ou seja a versão do html que será usada dna página
  • <html >:A tag html é o elemento principal do documento html, ela defini o container raiz de toda página
  • <head >:A tag head contém os metadados que serão interpretados pelo navegador. Nela encontraremos o título da pagina e links arquivos para configuração do documento
  • <body >:a tag body é responável pela exibição do conteúdo visível do documento. ou seja , tudo que estiver neste bloco será visível para o usuários
