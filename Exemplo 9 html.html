<header>

Exemplo sobre Web Semântica

Este bloco refere-se a tag semântica <header> que representa o cabeçalho de uma página. Nele geralmente utilizaramos a tag de seção/titulo

<nav>

Neste bloco representamos a seção de nevegação que utiliza a tag <nav>. NEste bloco definimos por exemplo os menus de navegação do nosso documentos HTML. Pro exemplo:

  • Início
  • Cadastro
  • Relatórios
  • Movimentação
  • Sobre

<main>

A tag <main> utilizada neste bloco representa o conteúdo principal da página. Nele teremos as tags semânticas <section>, <narticle> e <aside>

HTML - Web Semântica

<section>

Neste bloco estamos usando a tag semântica <section> para definir uma seção genérica de um documento.

a web semântica refere-se ao uso do HTML e outras tecnologia para estruturar o conteúdo de uma página de forme que ela seja compreendida tanto por humano quanto por máquinas. Isso significa usar elementos HTML que descrevam o significado do conteúdo em vez de apenas sua aparência.

<article>

A tag semântica <article> define como um conteúdo independente, como uma postagem de blog ou notícia.

<aside>

A tag semântica <aside> representa um conteúdo auxilar relacionado, mas não essencial, ele muitas vezes é utilizado como um menu auxilar.

  • Menu auxiliar 1
  • Menu auxiliar 2
  • Menu auxiliar 3
  • Menu auxiliar 4

<footer>

A tag semântica footer é aquela que define um rodapé do documento HTML. Nela geralmente colocamos informçaões de fechamento do nosso documento, tais como nome, autor, data, etc...
