Exemplo 7 HTML - listas ordenadas <ol> e Listas não ordenadas <ul>

Este exemplo mostra como criar listas ordenadas e não ordenadas incluindo subitens(listas alinhadas).

listas ordenadas <ol>

As kistas ordenadas são aquelas que seguem uma ordem numérica ou por letras. São utilizadas para mostrar conteúdo html em formato de lista.

Exemplo da lista ordenada <ol> juntamente com os elementos dessa lista que é representado pela tag <li>

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4
  5. Item 5

Exemplo 2 de listas ordenadas

  1. Item 1
    1. subitem1.1
    2. subitem1.2
    3. subitem1.3
    4. subitem1.4
    5. subitem1.5
  2. Item 2
  3. Item 3
  4. Item 4
    1. subitem4.1
    2. subitem4.2
    3. subitem4.3
    4. subitem4.4
    5. subitem4.5
  5. Item 5

Listas não ordenadas - <ul>

As listas não ordenadas são aquelas que representam os elementos da lista por meio de bullets. Essas listas são muito utilizadas na criação de menus. ABaixp demonstramos alguns exemplos de listas não ordenadas.

Exemplo lista não ordenada

  • Item 1
  • Item 2
    • Item 2.1
    • Item 2.2
    • Item 2.3
      • 2.3.1
      • 2.3.2
      • 2.3.3
    • Item 2.4
  • Item 3
  • Item 4
  • Item 5
