O que é o Grid Layout

O Gird Layout é bidmencional, permitindo que o desenvolvedor possa separar o layout da página em linhas e colunas, diferentemente do flexbox que é unidimensional, ordenando os elementos em linha

Flexgrid ou Grid Layout

O LFexbox e o Grid Layout oidem ser usados em conjuntos. Você pode aplicar o FlexBox a elementos dentro de um Layout de grade, como o Grid Layout, aproveitando o melhor dos dois mundos

A escolha entre Flexbox e Grid Layout não se trata de um subtituir o outro, mas sim de entender as necessidades especifícas do seu layout e escolher a tecninca que melhor atenda a suas possibilidades
