Visão Geral sobre CSS

CSS é um acronimo para folhas de estilo em cascata. Essa técnica visa personalisar, estilizar um documento HTML Para isso, usamos regras CSS que representam uma estrutura que define como os elementos selecionados devem ser estilizados. Essa regra é composta pelas seguintes partes:

  • SeletorIndica quais elementos serão afetados pelo estilo definido
  • Bloco de declaraçãoÉ o bloco composto por propriedades e valores, esse bloco será aplicado ao elemento HTML.

Tipos de Seletores

Em CSS seletores são padrões usados para selecionar e estilizar os elementos HTML em uma página WEB. ELes defiinem quais elementos receberão as regras de estilo que serão declaradas dentro das chaves. Possuimos vários tipos de seletores os principais são:

Seletores de ID

O seletor de ID é aquele que aplica a regra css a um componete html que tenha um atributo id no componete html. Este seletor de id deve ser aplicado a um único elemento html, haja vista que o id deve ser único. Este seletor tem prioridade sobre e demais seletores(de classe e de tipo)

Este seletor sempre começara com um "#", seguido pelo nome do seletor e sua propriedades e valores.

Neste exemplo usamos seletores de id, de classe e de tipo para estilizar o documento html.
